# Changelog

All notable changes to the Giant Dragon Egg mod will be documented in this file.

## [1.0.0] - 2026-07-01

### Added
- Implemented the Giant Dragon Egg (2x2x2 blocks).
- Custom logic to dynamically clear out a 2x2x2 area upon placement.
- Perfectly scaled 32x32x32 bounding box and interaction hitboxes.
- Teleportation logic that moves the giant egg cleanly to a new location when attacked.
- Full gravity support; breaks into an item if falling on non-solid blocks.
- Smart dimension restriction: only returns to 1x1 normal size when placed explicitly on Bedrock inside The End dimension.
- Forge 1.21, NeoForge 1.21, and Fabric 1.21 support.
